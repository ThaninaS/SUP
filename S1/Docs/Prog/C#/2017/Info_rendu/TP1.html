BONUS: my sl

The bonus consists of reproducing the behavior of the sl command from your terminal.
To test it, open a terminal and run:

while [ 1 -eq 1 ]; do sl; done

﻿﻿Read the sl manpage for more information.

Your goal is to implement the following function:

public static void MySl();

Be creative, it doesn't have to be a train.
